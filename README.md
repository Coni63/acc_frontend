# Installation

## Create environment

```sh
npm install
```

# Running

## basic one

```sh
ng serve --o
```

## using proxy

```sh
ng serve --proxy-config proxy.conf.json --o
```

or (this was set in package.json)

```sh
npm start
```

**INFO**: if you want to share it on network, add `--host 0.0.0.0`
